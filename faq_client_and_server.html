Code Generation

How can I avoid the plugin to generate the code, when the GraphQL schema(s) has not changed?

Since release 1.12.2, the plugin contains a skipGenerationIfSchemaHasNotChanged parameter. The default value is set to false , as this parameter is currently an incubating feature. Its default value will be changed to true starting from 2.x releases. See the plugin doc for more information.
