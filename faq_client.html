How can I generate a client that can attack several GraphQL servers?

In order to generate the source code for several GraphQL servers, you'll first need to separate the GraphQL schema in one folder par GraphQL server. This folder would typically be src/main/GraphQLServerName1 . Then, you'll create an execution of the plugin for each such folder, like this: 

					



						
							com.graphql-java-generator
							graphql-maven-plugin
							
								
									graphql-generateCode
									
										generateClientCode
									
									

										... Plugin parameters, for instance :

										org.allGraphQLCases.client
										src/main/GraphQLServerName1
										true
										true
									
								
								
									

										... Configuration for the src/main/GraphQLServerName2

Up to 1.12.2, in order to use these two GraphQL servers, you'll have to use the non spring app, as described here
